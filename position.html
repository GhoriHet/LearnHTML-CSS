Position Property

Static

element with static property can not move its position of left/right/top/bottom.

Relative Property

element with relative property can move its position of left/right/top/bottom.

Absolute WithOut Parent

element with Absolute position move accroding to body.
element move out from its normal flow.
element move with scroll..



Absolute With Parent

absolute block move accroding to their parent.

Fixed

element with fixed position move accroding to body and can not scrolled.

Sticky

Relative + Fixed

Intially work as a relative, whenever their condition (top: 10px) satisfied it work as a fixed.
